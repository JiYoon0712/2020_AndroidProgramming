> 핵심 서비스 
    
 ![image](https://user-images.githubusercontent.com/98688979/205251967-cbbdaf99-2efc-4a64-823b-279ed428e8e6.png)
- 메인 이동 클릭 시, 아래 4가지 서비스를 이용할 수 있다.
        1. 소개 
        2️. 포스터 
        3️. 평가/리뷰 
        4️. 부가정보
